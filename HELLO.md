# Hello World Demo

This is a simple starter demo for the Auto Claude framework.

## Purpose

Demonstrates the basic workflow of the Auto Claude autonomous coding framework by creating a minimal "Hello World" example.

## Usage

Run the script:
```bash
python hello_world.py
```

Expected output:
```
Hello, World!
```

## What This Demonstrates

1. **Spec Creation**: How to define a feature specification
2. **Implementation Plan**: Breaking down work into phases and chunks
3. **Execution**: Running the implementation through the framework

## Files

- `hello_world.py` - The main script
- `HELLO.md` - This documentation
- `.auto-claude/specs/demo-hello-world/` - The spec directory
